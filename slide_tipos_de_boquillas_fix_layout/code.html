Tipos de boquillas para oxiacetileno

Pequeña
1/8" - 1/4"

Trabajo Fino

Ideal para láminas delgadas y soldadura de precisión en joyería o componentes electrónicos industriales.

Mediana
3/8" - 1"

Uso General

Estándar para mantenimiento industrial, reparación de marcos y estructuras metálicas de calibre medio.

Grande
1" - 2"

Servicio Pesado

Diseñada para cortes profundos y precalentamiento de placas de acero estructural masivo.

warning Precaución Técnica

El exceso de presión en boquillas pequeñas puede causar retroceso de llama. Siempre verifique la tabla de presiones del fabricante antes de iniciar la ignición.

info

Mientras mayor sea el espesor , mayor sera la boquilla y mayor el flujo requerido de oxigeno.
